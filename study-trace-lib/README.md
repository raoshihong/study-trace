1.先将相关数据操作日志变更存储起来
2.再将数据进行计算(根据相关配置进行计算),计算完后再存储
3.进行相关的报表导出 (配置那张表导出哪些相关数据字段,策略模式,在查询是根据配置获取(根据哪张表))

数据变更情况：
1. 单表单记录变更
2. 单表多记录变更
2. 多表变更 (比如渠道停用或删除,导致客户经理也跟着停用或删除), (排序: 排序这个字段在子表中去了) ,需要画sql,将对应的数据管理查询出来作为标识数据
